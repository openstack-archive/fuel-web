Welcome to Fuel Installer

Help us to improve your experience by sending Mirantis information about the settings, features, and deployment actions when you use Fuel.

Usage statistics include information such as settings, button/menu clicks, hardware configuration, and version information. The usage statistics do not include any sensitive information such as passwords, ip addresses, or node names. For a complete list of statistics that we gather, click here.

Usage statistics include the following. Sensitive information such as passwords, ip addresses, and node names are not included.

Actions:

  • Operation type (adding cluster, adding node, deployment, removing node, e.t.c.)
  • Operation start and finish time
  • Actual time that is took to complete the operation
  • Network verification - whether it was used, and what was the result
  • Is OSTF used, and tests results

Environment Settings:

  • Number of environments
  • Distribution / OS
  • Network type
  • Kernel parameters
  • Admin network parameters
  • PXE parameters
  • DNS parameters
  • Storage options
  • Related Projects
  • Settings modified on Settings tab
  • Networking configuration

Node Settings:

  • Number of nodes deployed
  • Roles deployed to each node
  • Disk layout
  • Interfaces configuration

System Info:

  • Hypervisor
  • Hardware info
  • Fuel version info
  • OpenStack version info
